# RenPy-Story-Graph-Visual-Editor
一个可视化的 Ren'Py 剧情流程设计工具，支持拖拽式创建对话章节（label）、跳转/调用（jump/call）、菜单选项以及变量操作。
内置**变量区间分析**和**可达性检查**，能够自动发现不可达节点、条件分支失效等逻辑漏洞，并直接导出为可运行的 `.rpy` 脚本。
> **声明**：本文件部分代码由 AI 辅助生成，主体逻辑与设计由原作者完成。
## 主要功能

- **图形化编辑**  
  拖拽添加节点、拖拽连线创建剧情跳转（支持 jump / call），支持无条件边、条件分支边、菜单选项边。

- **变量操作与区间分析**  
  支持赋值、增强赋值（+=、-= 等）、复杂表达式和部分内置函数（abs、min、max）。  
  基于区间抽象解释自动推导每个节点的变量取值范围，检测不可达节点、下落风险（所有分支条件均为假）。

- **导入/导出 Ren'Py 脚本**  
  读取现有 `.rpy` 文件自动生成节点和边；导出时自动生成 `_control` 标签层级结构，保留变量操作和分支逻辑。

- **历史撤销/重做**（支持 Ctrl+Z / Ctrl+Y）  
- **保存/加载布局**（JSON 格式）  
- **详细分析报告** – 可保存为纯文本文件

### 环境要求
- Python 3.6 或更高版本
- 无需额外安装第三方库（仅使用标准库 tkinter、json、ast、math 等）

### 运行编辑器
python editor.py   # 请将文件名改为你保存的 .py 文件名

# 图形界面操作说明

以下表格列出了本编辑器的所有常用操作，你可以随时查阅。

| 编辑连线属性 | 双击连线，修改跳转类型、条件表达式或菜单文本 |

| 缩放画布 | 鼠标滚轮 |

## 连线类型说明

- **无条件跳转**：不弹菜单，也不判断条件，直接跳转。
- **条件分支**：根据条件表达式（如 `love >= 5`）的真假决定是否走该边。多个条件分支会生成 `if / elif` 结构。
- **菜单选项**：显示一个带有特定文本的菜单项，玩家选择后执行对应的 jump / call。可以附加条件（如 `if money > 100`）。

## 变量操作语法（节点内）

每行一条语句，支持以下形式：

- 简单赋值：`x = 10`
- 增强赋值：`y += 5` 、`score -= 2` 、`value *= 3`
- 复杂表达式：`result = (a + b) * 2 / c`
- 函数调用：`x = abs(y)` 、`m = min(a, b, c)`
- 逻辑表达式：`unlock = love > 5 and money >= 100`

## 条件表达式语法（连线上）

- 比较运算：`age >= 18`、`score < 60`
- 逻辑组合：`condition1 and condition2` 、`love > 5 or trust > 3`
- 变量比较：`x != y`、`a == b`
- 可包含函数调用：`abs(score) > 10`

> 注意：不支持 `elif` 或嵌套，编辑器内部自动根据多条边的条件顺序生成 `if-elif-else` 结构。
